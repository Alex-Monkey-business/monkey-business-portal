# Quick usage

## Plain HTML
```html
<img src="/brand/logos/amb-monogram.svg" alt="AMB logo" />
<img src="/brand/logos/monkey-freerider.svg" alt="Monkey Freerider logo" />
```

## Favicon
```html
<link rel="icon" href="/brand/icons/favicon.svg" type="image/svg+xml" />
<link rel="apple-touch-icon" href="/brand/icons/apple-touch-icon.png" />
```

## CSS color control
The `amb-monogram.svg`, `amb-monogram-tuft.svg`, `monkey-freerider.svg`, `monkey-freerider-wink.svg`, `monkey-minimal.svg`, and motion frames use `currentColor`.

```html
<div style="color:#1B1C1F">
  <img src="/brand/logos/amb-monogram.svg" alt="AMB" />
</div>
```
