# AMB brand assets

Version 1 av et lite brand kit bygget fra konseptretningen du landet på:
- AMB-monogram som hovedmerke
- freerider-ape som signaturmerke
- splash / motion frames
- favicon- og app-ikonfiler
- fargetokens for web

## Anbefalt bruk
- `logos/amb-monogram.svg` → navbar, footer, repo, små flater
- `logos/amb-monogram-tuft.svg` → mer leken variant
- `logos/monkey-freerider.svg` → splash, avatar, social, loading state
- `logos/monkey-freerider-wink.svg` → easter egg, playful states
- `icons/favicon.svg` → standard favicon
- `icons/app-icon-1024.png` → app icon / social avatar base

## Farger
- Ink: `#1B1C1F`
- Sand: `#EFE7DA`
- Pink: `#C85E87`
- Sun: `#EFC64C`
- Grape: `#9277BA`
- Snow: `#F8F4EC`

## Struktur
- `logos/` – masterlogoer i SVG
- `icons/` – favicon og appikoner
- `motion/splash-sequence/` – enkle frames for intro/splash
- `tokens/` – CSS/JSON farger
- `social-og-image.png` – rask preview til deling

## Tips
SVG-filene i `logos/` er de riktige masterfilene å bruke i repoet ditt. PNG-filene er mest for plattformer som trenger raster.
